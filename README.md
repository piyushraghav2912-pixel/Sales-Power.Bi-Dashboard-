# Sales Power BI Dashboard

This project is a *Sales Dashboard created using Microsoft Power BI* to analyze sales performance and business insights.

## Project Description
In this project, sales data is analyzed to understand total sales, profit, orders, and trends.  
The dashboard is interactive and helps in easy decision-making.

## Features
- Total Sales, Profit and Orders KPIs
- Category-wise and Region-wise sales analysis
- Sales trend analysis
- Interactive filters and slicers

## Dataset
The dataset used in this project is *SuperStore Sales Dataset*.

## Tools Used
- Microsoft Power BI
- Power Query
- DAX (Data Analysis Expressions)

## Files Included
- sales.project.pbit – Power BI file  
- SuperStore_Sales_Dataset.csv – Dataset  
- sales.project.png – Dashboard screenshot
